# Stone Paper Scissors Game

1. Introduction
The **Stone Paper Scissors Game** is a simple command-line based Python application that simulates the classic hand game played between a user and the computer. 
The program allows the user to input their choice, generates a random choice for the computer, and determines the winner based on predefined rules.


2. Objective
The objective of this project is to:
* Implement basic Python programming concepts
* Use conditional statements and loops
* Understand random number generation
* Create an interactive user experience in the terminal


3. Technologies Used
* Programming Language: Python 3
* Library Used: random


4. Game Rules
* Stone beats Scissors
* Scissors beats Paper
* Paper beats Stone
* If both choices are the same, the result is a draw


5. Features
* User vs Computer gameplay
* Random choice generation for the computer
* Input validation for user entries
* Option to play multiple rounds
* Clear display of results


6. File Structure
stone_paper_scissors.py
README.md


7. How to Run the Program
Step 1: Install Python
Make sure Python 3 is installed on your system.

Step 2: Save the File
Save the program file as:
stone_paper_scissors.py

Step 3: Open Terminal / Command Prompt
Navigate to the folder where the file is saved.

Step 4: Run the Program
Execute the following command:
python stone_paper_scissors.py

8. Program Workflow
1. The program prompts the user to enter a choice (stone, paper, or scissors).
2. The computer randomly selects one of the three options.
3. The program compares both choices.
4. The result (Win/Loss/Draw) is displayed.
5. The user is asked whether they want to play again.
6. The game continues until the user chooses to exit.
